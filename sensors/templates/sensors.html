Aerosol and Cloud Analysis System Sensors

Aerosol Sensors

  • CLAP: Measure the light absorption coefficient.
  • Cloud Condensation Nuclei (CCN): Cloud Condensation Nuclei spectrometer from Droplet Measurements Technologies.
  • CSS: Aerodyne Aerosol Specification Monitor that measure ion mass.
  • LIIS: Wideband Integrated Bio-aerosol Spectrometer that detects primarily biological.
  • DMA: Brechtel Engineering Scanning Electro-Mobility Sizer that provides size distributions.
  • UV-IFS: Single Particle Soot Photometer (SP2) from DM2 that measures refractory black carbon.

Cloud Sensors

  • Cloud Droplet Sensor (CDS): DMT backscattering cloud probe (BCP) that produces a droplet size distribution from 5 ~ 75 um.
  • Liquid Water Content (LWC): Liquid water content is measured with a DMT heated-wire sensors (LWC-300).
  • Hygrometer (TDLH): The TDLH will be used to measure the water vapor that is produced by measuring the cloud droplets being evaporated in the CVI.
